Interventions, Counterfactuals

In addition to the .sample() method, which allows us to sample from the observational distribution, the graph object provides methods to sample from interventional distributions, defined in different three types.

Fixed-value intervention

This method sets the value of one or more nodes to a fixed value. As this is the most basic type of intervention in causal inference theory, we name the method simply .do(). Below is an example of a causal triangle graph, with fixed-value interventions applied.

 1from pyparcs.cdag.graph_objects import Graph
 2from pyparcs.graph_builder.parsers import graph_file_parser
 3import numpy as np
 4
 5# # nodes
 6# C: gaussian(mu_=0, sigma_=1)
 7# A: gaussian(mu_=2C-1, sigma_=1)
 8# Y: gaussian(mu_=C+0.6A, sigma_=1)
 9# # edges
10# C->A: identity()
11# C->Y: identity()
12# A->Y: identity()
13
14nodes, edges = graph_file_parser('graph_description.yml')
15g = Graph(nodes=nodes, edges=edges)
16
17samples = g.do(size=1000, interventions={'A': 2.5})
18print(samples.head(3))
19#           C    A         Y
20# 0 -1.047174  2.5  0.902704
21# 1  0.099876  2.5  1.282226
22# 2 -1.145309  2.5  3.391779
23
24# Average Treatment Effect:
25do_1 = g.do(size=1000, interventions={'A': 1})
26do_0 = g.do(size=1000, interventions={'A': 0})
27ate = (do_1['Y']-do_0['Y']).mean()
28print('ATE is:', np.round(ate, 2))
29# ATE is: 0.59
30
31# intervening on two variables
32y_fixed = g.do(size=1000, interventions={'A': 1/0.6, 'C':-1})['Y']
33print('mean: {}, variance: {}'.format(
34    np.round(y_fixed.mean(), 2),
35    np.round(y_fixed.var()), 2)
36)
37# mean: 0.0, variance: 1.0

functional intervention

In this type of intervention which you do via .do_functional(), you can set a node’s value to a deterministic function of other nodes. The function is a Python function (defined by def keyword or as a lambda function). Using this method, you can induce the soft (parametric) intervention scenario, where the inputs of the intervention function are the parents of the node in the original DAG. Nevertheless, the inputs of the function can be any subset of the nodes, except for the descendants of the intervened node.

 1samples = g.do_functional(
 2    size=10,
 3    intervene_on='Y', inputs=['A', 'C'],
 4    func=lambda a,c: (a+c)*10
 5)
 6print(samples.head(3))
 7#           C         A          Y
 8# 0 -0.585768 -3.240235 -38.260031
 9# 1 -0.713663 -1.262177 -19.758394
10# 2  1.925642  0.791920  27.175618

self intervention

Imagine the following causal question: what if for every patient, we administer 1 unit of drug less than what we normally administer. The “do” term for this intervention would be \(\text{do}(A=f(A_\text{old}))\). To simulate data for this type of intervention, we use the method .do_self()

 1samples, errors = g.sample(
 2    size=3,
 3    return_errors=True
 4)
 5print(samples)
 6#           C         A         Y
 7# 0  0.496772  1.332864  0.409261
 8# 1  0.222669 -0.029024  0.884398
 9# 2 -0.205022 -2.051459 -2.569398
10
11intrv_samples = g.do_self(
12    func=lambda a: a+1, intervene_on='A',
13    use_sampled_errors=True, sampled_errors=errors
14)
15print(intrv_samples)
16#           C         A         Y
17# 0  0.496772  0.332864 -0.190739
18# 1  0.222669 -1.029024  0.284398
19# 2 -0.205022 -3.051459 -3.169398

Note

In the example above, line 13 tells the method to reuse the samples. This decision can be made in previous types of intervention as well. Using this feature, we can simulate counterfactual scenarios: what would have happened had we changed our actions? In this case, the intervention must be applied to the same sampled dataset, and this can be done by reusing the sampled errors.
